# โปรแกรมจัดการออร์เดอร์ร้านขายเครื่องดื่ม

โปรแกรมสำหรับการจัดการออร์เดอร์เครื่องดื่ม โดยมีระบบ ดังนี้
- การสั่งซื้อจนถึงการพิมพ์ใบเสร็จ
- จัดการและเปลี่ยนสถานะของออร์เดอร์ (กำลังดำเนินการ, เสร็จสิ้น, ยกเลิก)
- ระบบสมาชิกเบื้องต้น (ใช้ชื่อและเบอร์โทรศัพท์)
- บันทึกข้อมูลผ่านการใช้ JDBC ในการเชื่อมต่อระหว่าง Java และ MySQL
